#+title: mysql

[[https://melpa.org/#/mysql][file:https://melpa.org/packages/mysql-badge.svg]]

A pure Emacs Lisp MySQL wire protocol client.  It speaks the
classic MySQL protocol directly and does not shell out to the =mysql= CLI.

* Features

- TLS connections
- Authentication with =mysql_native_password=
- Authentication with =caching_sha2_password=
- Authentication with =sha256_password=
- Prepared statements
- Transactions
- Public transaction helpers: =mysql-autocommit-p=, =mysql-in-transaction-p=,
  =mysql-set-autocommit=, =mysql-commit=, and =mysql-rollback=
- Public connection state helpers: =mysql-live-p=, =mysql-busy-p=,
  =mysql-connection-id=, =mysql-current-database=, and =mysql-select-database=
- Public response drain helper for clients that cancel a running query out of
  band: =mysql-drain-query-response=
- Support for MySQL 5.6+, MySQL 8.x, and MariaDB 10.11+

* Usage

#+begin_src emacs-lisp
(require 'mysql)

(let* ((conn (mysql-connect :host "127.0.0.1"
                                 :port 3306
                                 :user "root"
                                 :password "secret"
                                 :database "mydb"))
       (result (mysql-query conn "SELECT 1 AS n")))
  (unwind-protect
      (message "Rows: %S" (mysql-result-rows result))
    (mysql-disconnect conn)))
#+end_src

Transaction helpers are available when you want explicit control over the
session state:

#+begin_src emacs-lisp
(mysql-set-autocommit conn nil)
(unwind-protect
    (progn
      (mysql-query conn "INSERT INTO demo(name) VALUES ('alice')")
      (mysql-commit conn))
  (mysql-set-autocommit conn t))
#+end_src

* Installation

Install =mysql= from MELPA:

#+begin_src emacs-lisp
(package-install 'mysql)
#+end_src
